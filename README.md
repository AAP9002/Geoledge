## Set up
```bash
npm install

cd client

npm install
```
- make a .env file with contents from .env.TEMPLATE with actual passwords replaced in your .env file

## Boot Up
Start backend
```bash
node server.js
```
Start front end
```bash
cd client

npm start
```
